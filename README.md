# 🚀 Employee Management System (EMS)

A comprehensive, full-stack **Java-based web application** designed to automate and streamline **HR and employee management operations**. This system provides a secure, role-based platform for managing organizational resources effectively.

---

## 🌟 Key Features

### 👔 HR Admin (Manager) Module
The HR Admin has centralized control over organizational operations:
- **📊 Interactive Dashboard**: Real-time statistics with visual data representation.
- **👥 Employee Management**: Full CRUD operations for employee records.
- **🏢 Department Management**: Create and organize company departments.
- **⏱️ Attendance Tracking**: Monitor attendance across the entire organization.
- **📄 Leave Management**: Review and process (Approve/Reject) leave requests.
- **💰 Payroll & Salaries**: Automated payroll management and digital salary slip generation.
- **📢 Notice Board**: Broadcast company-wide announcements.
- **🎯 Recruitment Pipeline**: End-to-end tracking (Applied → Interview → Selected).
- **🪪 ID Card Generation**: Instantly generate and print employee ID cards.

### 👤 Employee (Staff) Module
Self-service portal for employees to manage their work life:
- **🏠 Personal Dashboard**: Individual performance stats and latest company news.
- **📅 My Attendance**: View personal attendance history.
- **📩 Leave Application**: Submit leave requests and track approval status.
- **💸 My Salary**: Access salary history and download digital slips.
- **🆔 Digital ID Card**: View and print official employee ID.
- **🔔 Announcements**: Stay updated with company-wide notices.

## ⚙️ Installation & Setup

1. **Prerequisites**: Ensure you have XAMPP/WAMP (Apache & MySQL) and a Java IDE (IntelliJ/Eclipse) or Maven installed.
2. **Database Setup**:
   - Start MySQL and open phpMyAdmin.
   - Create a database named `ems_db`.
   - Import the provided SQL script (if available).
3. **Clone & Run**:
   ```bash
   git clone https://github.com/YOUR_USERNAME/EMS_Final.git
   ```
4. **Deploy**:
   - Open the project in your IDE.
   - Configure your server (e.g., Tomcat).
   - Run the application.
